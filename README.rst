======
tworld
======

Unofficial T world API for Python.


Installation
------------

.. code-block:: console

   $ pip install tworld


Getting Started
---------------

Login
~~~~~

.. code-block:: python

   tworld = TWorld()
   tworld.login('username', 'password')
   print tworld.logged_in()  # True
